# 30台電腦部署 - 快速開始指南

> 5分鐘快速了解部署流程

---

## 🎯 部署目標

在30台電腦上部署項目，使用 Fine-Tuned 模型，**確保所有數據完全隔離，不流出本地**。

---

## 📦 部署流程（3個步驟）

### 主控電腦（準備部署包）

```bash
# 1. 導出模型
ollama save anti-fraud-expert-20251111_220455 -o expert_model.tar
ollama save anti-fraud-scammer-20251111_220455 -o scammer_model.tar
ollama save gemma3:4b -o gemma3_4b_base.tar

# 2. 創建部署包
scripts\create_deployment_package.bat

# 3. 分發到30台電腦（USB/內網）
```

### 每台目標電腦（3個命令）

```bash
# 1. 安裝模型
scripts\install_models.bat

# 2. 配置環境
scripts\setup_environment.bat

# 3. 驗證部署
scripts\verify_deployment.bat

# 4. 啟動服務
cd code
本地启动.bat
```

---

## 🔒 數據安全關鍵配置

### 必須檢查的配置

在每台電腦的 `backend\.env` 文件中：

```bash
# 關鍵配置（必須）
OLLAMA_BASE_URL=http://127.0.0.1:11434  # 只監聽本地
OFFLINE_MODE=true                        # 啟用離線模式
DISABLE_EXTERNAL_APIS=true              # 禁用外部 API
DISABLE_DATA_UPLOAD=true                # 禁用數據上傳

# 模型配置
AGENT_MODEL_EXPERT=anti-fraud-expert-20251111_220455
AGENT_MODEL_SCAMMER=anti-fraud-scammer-20251111_220455
AGENT_MODEL=gemma3:4b
```

## ✅ 快速驗證

### 檢查模型
```bash
ollama list | findstr anti-fraud
```

### 檢查網絡
```bash
netstat -an | findstr "11434"
# 應該只看到 127.0.0.1:11434
```

### 檢查配置
```bash
type backend\.env | findstr "127.0.0.1"
type backend\.env | findstr "OFFLINE_MODE"
```

## ⚠️ 重要提醒

1. **斷開互聯網**：部署和運行時確保電腦不連接互聯網
2. **本地模式**：Ollama 必須只監聽 `127.0.0.1`，不能是 `0.0.0.0`
3. **離線模式**：確認 `OFFLINE_MODE=true` 已設置
4. **數據存儲**：所有訓練數據存儲在本地 `backend/training_data/`
