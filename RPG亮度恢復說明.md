## 💡 如需再次調整亮度

### 臨時調整（瀏覽器）

在瀏覽器控制台（F12）運行：

```javascript
// 提高亮度
document.body.style.filter = 'brightness(1.5)';

// 降低亮度
document.body.style.filter = 'brightness(0.8)';

// 恢復預設
document.body.style.filter = '';
```

---

### 永久調整（修改文件）

如果以後想調整亮度，在相應的HTML文件中添加：

```html
<body style="background-color: black">
    <style>
        body {
            /* 調整數值：1.0 = 預設，1.5 = 亮50%，0.8 = 暗20% */
            filter: brightness(1.3);
            -webkit-filter: brightness(1.3);
        }
    </style>
    ...
</body>
```

## 🎯 建議的亮度值

根據不同需求，可以選擇不同的亮度：

| 亮度值 | 效果 | 適用場景 |
|--------|------|----------|
| **0.7** | 較暗 | 夜間遊玩、護眼 |
| **0.8** | 稍暗 | 減少刺眼感 |
| **0.9** | 略暗 | 微調 |
| **1.0** | 預設 ✅ | 原版體驗（當前） |
| **1.1** | 略亮 | 微調 |
| **1.2** | 稍亮 | 增加可見度 |
| **1.3** | 較亮 | 明亮環境 |
| **1.5** | 很亮 | 最大可見度 |
