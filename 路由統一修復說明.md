## 🔧 修改內容

### 1. 後端路由修改 (`backend/main.py`)

#### ✨ 新增：根路徑返回主頁

**之前**：
```python
@app.get("/")
def read_root():
    return {"status": "Backend is running", ...}
```

**現在**：
```python
@app.get("/")
def read_root():
    """主頁路由 - 返回模式選擇頁面"""
    index_path = os.path.join(os.path.dirname(__file__), '..', 
                               'RPG_platform', 'RPG_Project', 'index.html')
    if os.path.exists(index_path):
        return FileResponse(index_path)
    else:
        return {"status": "Backend is running", ..., "error": "index.html not found"}

@app.get("/health")
def health_check():
    """健康檢查端點"""
    return {"status": "Backend is running", ...}
```

**好處**：
- ✅ 訪問 `http://localhost:8000/` 直接顯示主頁
- ✅ 健康檢查移到 `/health`
- ✅ 用戶體驗更好

### 2. 前端鏈接智能調整 (`RPG_Project/index.html`)

#### ✨ 新增：環境檢測和自動鏈接調整

```javascript
// 環境檢測並自動調整所有鏈接
const isHTTP = window.location.protocol === 'http:' || 
               window.location.protocol === 'https:';

window.addEventListener('DOMContentLoaded', function() {
    console.log(`環境: ${isHTTP ? 'HTTP服務器' : '本地文件 (RPG Maker)'}`);
    
    // 如果是HTTP環境，調整所有鏈接為絕對路徑
    if (isHTTP) {
        // RPG遊戲
        document.querySelectorAll('a[href="rpg_game.html"]').forEach(link => {
            link.href = '/RPG_Project/rpg_game.html';
        });
        
        // 自動模擬
        document.querySelectorAll('a[href="test_ai_agent_v2.5.html"]').forEach(link => {
            link.href = '/RPG_Project/test_ai_agent_v2.5.html';
        });
        
        // API測試
        document.querySelectorAll('a[href="test_plugin.html"]').forEach(link => {
            link.href = '/RPG_Project/test_plugin.html';
        });
        
        console.log('✅ 已調整鏈接為HTTP路徑');
    } else {
        console.log('✅ 使用本地相對路徑');
    }
});
```

**工作原理**：
- 檢測當前環境（file:// 或 http://）
- HTTP環境：自動添加 `/RPG_Project/` 前綴
- 本地環境：保持相對路徑

### 3. 個人對話模式跳轉頁面 (`personal_chat_redirect.html`)

#### ✨ 極簡跳轉頁面

```html
<!DOCTYPE html>
<html lang="zh-TW">
<head>
    <meta charset="UTF-8">
    <title>正在跳轉...</title>
    <script>
        const isHTTP = window.location.protocol === 'http:' || 
                       window.location.protocol === 'https:';
        if (isHTTP) {
            // HTTP環境：跳轉到個人對話頁面
            window.location.replace('/personal_chat.html');
        } else {
            // 本地環境：嘗試打開HTTP地址
            window.location.replace('http://localhost:8000/personal_chat.html');
        }
    </script>
</head>
<body>
    <!-- 立即跳轉，不顯示任何內容 -->
</body>
</html>
```

**特點**：
- ⚡ 立即跳轉，無延遲
- 🔇 不顯示中間頁面
- 🎯 自動環境適配
