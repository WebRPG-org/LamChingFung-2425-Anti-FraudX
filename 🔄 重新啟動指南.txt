┌─ ⚠️  如果仍然無法連接 ──────────────────────────────────────────┐
│                                                                     │
│  問題 1: 端口被占用                                               │
│  檢查: netstat -ano | findstr :8000                               │
│  解決: 找到 PID 並結束進程: taskkill /PID <PID> /F               │
│                                                                     │
│  問題 2: 防火牆阻擋                                               │
│  解決: Windows 防火牆 → 允許應用 → Python                        │
│                                                                     │
│  問題 3: Ollama 未運行                                            │
│  檢查: curl http://localhost:11434/api/tags                       │
│  解決: 執行 ollama serve                                          │
│                                                                     │
│  問題 4: 依賴問題                                                 │
│  解決: 重新安裝依賴                                               │
│        cd backend                                                  │
│        pip install -r requirements.txt                             │
│                                                                     │
