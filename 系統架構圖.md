## API路由映射

```
前端路由
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
GET  /                    → 主頁 (RPG_Project/index.html) ✨
GET  /rpg                 → RPG入口 (同主頁)
GET  /modes               → 模式選擇頁 (frontend/index.html)
GET  /personal_chat.html  → 個人對話 (frontend/personal_chat.html) ✨
GET  /test                → 測試頁面 (frontend/test.html)

靜態文件
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
GET  /RPG_Project/*       → 靜態資源 (HTML/JS/CSS/IMG) ✨

遊戲API
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
POST /api/game/start      → 創建遊戲會話 ✨
POST /api/game/message    → 生成AI回應 ✨
WS   /ws/game/{id}        → 實時通信 ✨

模擬API
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
POST /simulation/start    → 開始三方模擬 ✨
WS   /ws/simulation/{id}  → 實時對話流 ✨
POST /simulation/stop/{id}→ 停止模擬 ✨

個人對話API
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
POST /api/personal-chat/start       → 開始會話 ✨
POST /api/personal-chat/message     → 發送消息 ✨
DELETE /api/personal-chat/session/{id} → 結束會話 ✨
```

## 數據流程圖

### RPG遊戲模式

```
[玩家] 
  ↓ 點擊NPC
[RPG界面 (rpg_game.html)]
  ↓ WebSocket
[後端 Game API v2]
  ↓ 調用
[VictimAgent]
  ↓ LLM請求
[Ollama/Gemma]
  ↓ 生成回應
[ExpertAgent 分析]
  ↓ WebSocket返回
[RPG界面顯示對話]
  ↓
[玩家看到回應]
```

### 自動模擬模式

```
[用戶配置]
  ↓ 選擇人設和騙局
[點擊開始模擬]
  ↓ POST /simulation/start
[後端創建會話]
  ├─ ScammerAgent 初始化
  ├─ VictimAgent 初始化
  └─ ExpertAgent 初始化
  ↓ WebSocket連接
[自動生成對話 (8-12輪)]
  ├─ 騙徒說話
  ├─ 受害者回應
  └─ 專家評論
  ↓ 實時推送
[前端顯示三方對話]
  ↓ 模擬結束
[顯示分析報告]
```

### 個人對話模式

```
[用戶選擇模式]
  ├─ 🛡️ 防詐助手
  └─ 🎭 騙徒模擬 (選騙局類型)
  ↓ POST /api/personal-chat/start
[創建會話]
  ↓ 獲取session_id
[用戶輸入消息]
  ↓ POST /api/personal-chat/message
[AI Agent生成回應]
  ├─ ExpertAgent (助手模式)
  └─ ScammerAgent (騙徒模式)
  ↓ 返回回應
[前端顯示對話]
  ├─ 文字顯示
  └─ 語音播放 (TTS)
  ↓
[繼續對話或結束]
```
