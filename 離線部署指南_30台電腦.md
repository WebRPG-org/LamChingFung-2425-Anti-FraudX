## 📋 部署概述

### 目標
- ✅ 在30台電腦上部署項目

- ✅ **所有數據完全隔離，不流出本地**
- ✅ 統一配置管理

### 部署架構

```
主控電腦（1台）
├── 模型文件（.tar）
├── 項目代碼
```

## 💻 單機部署腳本（在30台電腦上運行）

### install_models.bat

```batch
@echo off
setlocal enabledelayedexpansion
chcp 65001 > nul

echo ========================================
echo 離線模型安裝腳本
echo ========================================
echo.

REM 檢查 Ollama 是否安裝
where ollama >nul 2>&1
if %errorlevel% neq 0 (
    echo [錯誤] Ollama 未安裝
    echo 請先安裝 Ollama: https://ollama.com
    pause
    exit /b 1
)

echo [1/3] 啟動 Ollama 服務...
start /B ollama serve
timeout /t 5 /nobreak >nul

echo [2/3] 導入模型...
if exist models\gemma3_4b_base.tar (
    echo    導入基礎模型: gemma3:4b
    ollama load models\gemma3_4b_base.tar
)

if exist models\expert_model.tar (
    echo    導入專家模型: anti-fraud-expert-20251111_220455
    ollama load models\expert_model.tar
)

if exist models\scammer_model.tar (
    echo    導入騙徒模型: anti-fraud-scammer-20251111_220455
    ollama load models\scammer_model.tar
)

echo [3/3] 驗證模型...
ollama list
echo.
echo ========================================
echo 模型安裝完成
echo ========================================
pause
```

## 🔒 數據隔離配置

### 確保數據不流出的配置

#### 1. 網絡隔離配置

在每台電腦的 `.env` 文件中：

```bash
# ============================================
# 網絡隔離配置（關鍵！）
# ============================================

# Ollama 只監聽本地
OLLAMA_BASE_URL=http://127.0.0.1:11434

# 禁用所有外部 API 調用
DISABLE_EXTERNAL_APIS=true

# 禁用數據上傳
DISABLE_DATA_UPLOAD=true

# 禁用 Hugging Face Hub
HF_HUB_OFFLINE=true
TRANSFORMERS_OFFLINE=1
```

#### 2. 防火牆規則（可選）

```batch
# block_external_network.bat
@echo off
echo 配置防火牆規則（阻止項目相關程序訪問互聯網）...

REM 阻止 Python 訪問外部網絡（可選，謹慎使用）
netsh advfirewall firewall add rule name="Block Python External" dir=out action=block program="C:\Python\python.exe" enable=yes

REM 阻止 Ollama 訪問外部網絡（模型已本地安裝）
netsh advfirewall firewall add rule name="Block Ollama External" dir=out action=block program="C:\Users\%USERNAME%\AppData\Local\Programs\Ollama\ollama.exe" enable=yes

echo 防火牆規則已配置
```

#### 3. 修改代碼確保離線運行

創建 `backend/utils/offline_mode.py`:

```python
"""離線模式檢查和強制"""
import os
import logging

log = logging.getLogger(__name__)

OFFLINE_MODE = os.getenv("OFFLINE_MODE", "true").lower() == "true"

def check_offline_mode():
    """檢查並強制離線模式"""
    if OFFLINE_MODE:
        # 禁用所有外部 API 調用
        os.environ["HF_HUB_OFFLINE"] = "1"
        os.environ["TRANSFORMERS_OFFLINE"] = "1"
        
        # 禁用 Hugging Face Hub
        try:
            from huggingface_hub import offline_mode
            offline_mode()
        except:
            pass
        
        log.info("🔒 離線模式已啟用 - 所有數據將保持在本地")
    else:
        log.warning("⚠️ 離線模式未啟用 - 請確認數據安全")

    """阻止外部請求（如果啟用離線模式）"""
```

## ⚠️ 重要注意事項

### 數據安全

1. **永遠不要配置外部 API**
   - 不要設置 Hugging Face Token
   - 不要設置雲端存儲配置
   - 不要啟用數據上傳功能

2. **網絡隔離**
   - Ollama 只監聽 `127.0.0.1`（本地）
   - 不監聽 `0.0.0.0`（所有接口）
   - 可選：配置防火牆規則

3. **數據存儲**
   - 所有訓練數據存儲在本地 `backend/training_data/`
   - 不配置任何雲端同步
   - 定期備份到安全位置（離線）

4. **日誌管理**
   - 日誌文件只存儲在本地
   - 不包含敏感信息
   - 定期清理舊日誌
