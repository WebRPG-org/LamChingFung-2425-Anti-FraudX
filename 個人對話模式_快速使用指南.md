## 🚀 推荐使用方式

### 方式1：纯文字模式（推荐）✅

**优点**：
- ✅ 完全可靠
- ✅ 无需担心网络
- ✅ 响应速度快

**操作**：
```
1. 直接在输入框打字
2. 按Enter或点击发送
3. AI自动语音播放回复
```

### 方式2：语音播放 + 键盘输入（平衡）🎵

**优点**：
- ✅ 解放眼睛（听AI回复）
- ✅ 输入可靠（键盘打字）
- ✅ 多感官体验

**设置**：
```
☑️ 保持「自動播放語音」勾选
⌨️ 使用键盘输入
```

---

### 方式3：完整语音模式（网络良好时）🎤

**要求**：
- ✅ 稳定的网络连接
- ✅ 可以访问Google服务
- ✅ 麦克风权限已允许

**操作**：
```
1. 点击麦克风图标 🎤
2. 说话（图标变红）
3. 自动填入文字
4. 点击发送
```

## 💡 使用技巧

### 文字输入技巧

1. **快速发送**
   ```
   按 Enter 键直接发送
   （无需点击发送按钮）
   ```

2. **清晰表达**
   ```
   ✅ "我收到短信说账户异常，要我点链接"
   ❌ "有人发消息"
   ```

3. **追问细节**
   ```
   AI回复后可以继续追问
   "为什么会这样？"
   "我应该怎么做？"
   ```

---

### 语音播放技巧

1. **控制播放**
   ```
   ☑️ 勾选：AI回复自动朗读
   ☐ 不勾选：纯文字模式
   ```

2. **最佳环境**
   ```
   🔇 安静环境听得更清楚
   🎧 使用耳机效果更好
   ```

3. **暂停播放**
   ```
   取消勾选「自動播放語音」即可
   ```

---

### 语音输入技巧（网络正常时）

1. **说话技巧**
   ```
   ✅ 清晰、正常语速
   ✅ 说完整句子
   ✅ 安静环境
   ❌ 太快或含糊
   ❌ 环境太吵
   ```

2. **检查识别**
   ```
   说完后查看输入框
   确认文字正确再发送
   必要时可以修改
   ```

3. **网络问题处理**
   ```
   如果出现network错误：
   1. 改用键盘输入
   2. 检查网络连接
   3. 关闭VPN重试
   ```
