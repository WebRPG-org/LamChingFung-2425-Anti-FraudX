## ✅ 測試清單

### A. 角色動畫測試

#### 測試 1: 單方向移動
- [ ] **向上移動** (W 或 ↑)
  - 動畫應該流暢播放 `walk-up`
  - 停止時切換到 `idle-up`
  - 不應該頻繁閃爍

- [ ] **向下移動** (S 或 ↓)
  - 動畫應該流暢播放 `walk-down`
  - 停止時切換到 `idle-down`

- [ ] **向左移動** (A 或 ←)
  - 動畫應該流暢播放 `walk-left`
  - 停止時切換到 `idle-left`

- [ ] **向右移動** (D 或 →)
  - 動畫應該流暢播放 `walk-right`
  - 停止時切換到 `idle-right`

#### 測試 2: 斜向移動 (重點測試)
- [ ] **右上移動** (W+D 或 ↑+→)
  - ✅ 應該保持 `walk-right` 動畫
  - ❌ 不應該在 `walk-up` 和 `walk-right` 之間切換

- [ ] **左上移動** (W+A 或 ↑+←)
  - ✅ 應該保持 `walk-left` 動畫

- [ ] **右下移動** (S+D 或 ↓+→)
  - ✅ 應該保持 `walk-right` 動畫

- [ ] **左下移動** (S+A 或 ↓+←)
  - ✅ 應該保持 `walk-left` 動畫

#### 測試 3: 方向切換
- [ ] 從向上切換到向右
  - 動畫應該立即切換，無延遲
  
- [ ] 從移動切換到靜止
  - 應該立即切換到 idle 動畫

---

### B. 地圖結構測試

#### 測試 4: Tilemap 顯示
- [ ] **地面層**
  - 應該看到草地 Tiles (綠色)
  - 草地應該有變化 (不是單一顏色)
  - 使用噪聲模式生成

- [ ] **道路系統**
  - 橫向主路: 3 tiles 寬，貫穿地圖
  - 縱向主路: 3 tiles 寬，貫穿地圖
  - 交叉路口: 中心有特殊裝飾 tile
  - 對角線小路: 連接四角到中心

#### 測試 5: 裝飾元素
- [ ] **樹木**
  - 地圖四邊應該有樹木
  - 每 4 tiles 一棵 (約 70% 生成率)
  - 不應該出現在道路上

- [ ] **花朵**
  - 地圖上隨機分佈約 50 個花朵
  - 不應該出現在主路上 (±2 tiles)

#### 測試 6: 地圖尺寸
- [ ] 地圖大小: 2400x2400 pixels (50x50 tiles)
- [ ] 相機跟隨角色移動
- [ ] 角色不能移出地圖邊界
- [ ] 相機縮放: 1.0x (標準視角)

### C. NPC 位置測試

#### 測試 7: NPC 分佈
- [ ] **城鎮中心** (4 個 NPC)

  - 分佈在主路周圍

- [ ] **外圍區域** (2 個 NPC)

  - 在次要道路附近

- [ ] **四角區域** (4 個 NPC)

  - 位置: (300, 300), (2100, 300), (300, 2100), (2100, 2100)

#### 測試 8: NPC 互動
- [ ] 靠近 NPC 時顯示互動提示
- [ ] 按 E 鍵可以互動
- [ ] NPC 有 AI 漫遊行為
- [ ] NPC 名稱標籤正確顯示

## 🔧 除錯技巧

### 查看動畫狀態
在瀏覽器 Console 輸入:
```javascript
// 查看當前動畫
game.scene.scenes[1].player.sprite.anims.currentAnim.key

// 查看角色方向
game.scene.scenes[1].player.currentDirection

// 查看移動狀態
game.scene.scenes[1].player.isMoving
```

### 查看地圖資訊
```javascript
// 查看地圖尺寸
const scene = game.scene.scenes[1];
console.log('Map size:', scene.map.width, 'x', scene.map.height);
console.log('Pixel size:', scene.map.widthInPixels, 'x', scene.map.heightInPixels);

// 查看圖層
console.log('Layers:', scene.map.layers.map(l => l.name));
```

### 查看 NPC 位置
```javascript
const scene = game.scene.scenes[1];
scene.npcs.forEach((npc, i) => {
  console.log(`NPC ${i}:`, npc.sprite.x, npc.sprite.y, npc.type);
});
```
