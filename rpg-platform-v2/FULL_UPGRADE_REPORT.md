## 🎯 使用指南

### 啟動遊戲

```bash
# 1. 確保素材已複製（如果還沒有）
.\copy-rpg-assets.ps1

# 2. 啟動開發伺服器
npm run dev

# 3. 打開瀏覽器
http://localhost:3000
```

### 遊戲操作

| 按鍵 | 功能 |
|------|------|
| **↑↓←→** 或 **WASD** | 移動角色 |
| **E** | 與 NPC 互動 |
| **1** | 切換為受害者角色 |
| **2** | 切換為騙徒角色 |
| **3** | 切換為專家角色 |
| **ESC** | 返回主選單 |

### 開發指南

#### 添加新 NPC

```typescript
// 在 WorldMapScene.ts 的 createNPCs() 中
const npcData = [
  { x: 400, y: 300, type: 'elderly' },
  // 添加新 NPC
  { x: 500, y: 400, type: 'student' }
];
```

#### 添加新動畫

```typescript
// 在 BootScene.ts 的 createCharacterAnimations() 中
this.anims.create({
  key: 'custom-animation',
  frames: this.anims.generateFrameNumbers('sprite-key', { 
    start: 0, 
    end: 5 
  }),
  frameRate: 10,
  repeat: -1
});
```

#### 修改地圖

```typescript
// 在 WorldMapScene.ts 的 createTilemapWorld() 中
// 修改地圖大小
this.map = this.make.tilemap({
  tileWidth: 48,
  tileHeight: 48,
  width: 50,  // 修改寬度
  height: 50  // 修改高度
});
```

## 🔧 已知問題與解決方案

### 問題 1: 素材未載入

**症狀**: 遊戲顯示黑屏或錯誤

**解決方案**:
```bash
# 確認素材已複製
ls public/assets/characters/

# 重新複製素材
.\copy-rpg-assets.ps1

# 清除快取並重啟
npm run dev
```

### 問題 2: 動畫不流暢

**症狀**: 角色移動卡頓

**解決方案**:
- 檢查 FPS（應為 60）
- 降低粒子效果數量
- 減少同時顯示的 NPC 數量

### 問題 3: 記憶體使用過高

**症狀**: 瀏覽器變慢

**解決方案**:
- 使用較小的地圖尺寸
- 減少裝飾物數量
- 定期清理未使用的精靈
