## 🎮 功能演示

### 遊戲流程

```
啟動遊戲
    ↓
載入資源（BootScene）
    ↓
主選單（MainMenuScene）
    ├─→ 選擇角色 → 世界地圖（WorldMapScene）
    │                    ↓
    │              移動並尋找 NPC
    │                    ↓
    │              按 E 互動 → 對話場景（BattleScene）
    │                              ↓
    │                        進行對話
    │                              ↓
    │                        返回世界地圖
    │
    └─→ 自動模式 → 自動模式場景（AutoModeScene）
                        ↓
                   AI 自動運行模擬
                        ↓
                   查看統計數據
```

### 角色切換流程

```
玩家在世界地圖
    ↓
按 1 鍵 → 切換到受害者（藍色）
按 2 鍵 → 切換到騙徒（紅色）
按 3 鍵 → 切換到專家（綠色）
    ↓
HUD 即時更新顯示
    ↓
進入對話時使用當前角色
```
