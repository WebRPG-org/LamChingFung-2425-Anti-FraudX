## 🚀 使用流程

### 步驟 1: 打開生成器

專業地圖生成器已在瀏覽器中打開！

### 步驟 2: 配置參數

1. 選擇地圖尺寸（推薦：3200x2400）
2. 選擇細節等級（推薦：超高）
3. 選擇建築數量（推薦：20 棟）
4. 選擇裝飾密度（推薦：高）

### 步驟 3: 生成地圖

點擊 **「🎨 生成專業地圖」** 按鈕

等待 5-10 秒（取決於地圖大小）

### 步驟 4: 下載地圖

點擊 **「💾 下載地圖」** 按鈕

檔案名稱：`professional-world-map.png`

### 步驟 5: 安裝地圖

將下載的檔案移動到：
```
rpg-platform-v2/public/assets/maps/professional-world-map.png
```

或使用自動安裝腳本：
```powershell
.\move-assets.ps1
```

## 💡 使用技巧

### 最佳實踐

1. **首次使用**：從中等配置開始測試
2. **性能考慮**：巨型地圖需要更多記憶體
3. **細節平衡**：超高細節適合展示，高細節適合遊戲
4. **裝飾密度**：過高可能影響視覺清晰度

### 優化建議

1. **地圖分塊**：大地圖可以分成多個區域
2. **動態載入**：只載入玩家附近的區域
3. **LOD 系統**：遠處使用低細節版本
4. **紋理壓縮**：使用 PNG 壓縮工具
