## 🚀 使用步驟

### 方法 1：手動下載（推薦）

1. 在打開的瀏覽器頁面中，點擊 **「💾 下載所有素材」** 按鈕
2. 瀏覽器會自動下載 6 個 PNG 檔案到您的下載資料夾
3. 將檔案移動到對應位置：

```
rpg-platform-v2/public/assets/
├── sprites/
│   ├── player.png          ← 從下載資料夾移動到這裡
│   ├── npc-scammer.png     ← 從下載資料夾移動到這裡
│   ├── npc-expert.png      ← 從下載資料夾移動到這裡
│   └── npc-victim.png      ← 從下載資料夾移動到這裡
├── maps/
│   └── world-map.png       ← 從下載資料夾移動到這裡
└── ui/
    ├── dialogue-box.png    ← 從下載資料夾移動到這裡
    └── button.png          ← 從下載資料夾移動到這裡
```

### 方法 2：使用 PowerShell 腳本（自動化）

如果您已經下載了所有檔案到下載資料夾，可以運行：

```powershell
.\move-assets.ps1
```

這個腳本會自動將檔案從下載資料夾移動到正確的位置。

## 🔧 故障排除

### 問題：素材無法載入
**解決方案**：
1. 確認檔案名稱完全正確（區分大小寫）
2. 確認檔案在正確的資料夾中
3. 重新整理瀏覽器 (Ctrl+F5)

### 問題：下載的檔案找不到
**解決方案**：
1. 檢查瀏覽器的下載資料夾（通常是 `C:\Users\您的用戶名\Downloads`）
2. 確認瀏覽器沒有阻擋下載

### 問題：遊戲畫面是黑色的
**解決方案**：
1. 打開瀏覽器開發者工具 (F12)
2. 查看 Console 標籤是否有錯誤訊息
3. 確認 Vite 開發伺服器正在運行

## 🎨 自訂素材

如果您想使用自己的素材：

1. 保持相同的檔案名稱和尺寸
2. 使用 PNG 格式（支援透明背景）
3. 角色精靈圖建議尺寸：48x48 或 64x64
4. 地圖建議尺寸：800x600 或更大
