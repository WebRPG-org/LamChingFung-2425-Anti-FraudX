## 🔧 調試工具

### 查看碰撞標記
打開瀏覽器控制台（F12），輸入：
```javascript
// 查看當前位置的 tile 碰撞標記
const scene = game.scene.scenes[0];
const player = scene.player;
const x = Math.floor(player.sprite.x / 48);
const y = Math.floor(player.sprite.y / 48);
console.log(`Player at tile (${x}, ${y})`);

// 查看特定 tile 的標記
const tileId = scene.map.getTileAt(x, y).index;
const flags = scene.collisionSystem.getTileFlags(tileId);
console.log(`Tile ${tileId} flags: ${flags} (0x${flags.toString(16)})`);
```

### 顯示碰撞區域（調試模式）
```javascript
// 在場景中添加碰撞可視化
scene.physics.world.createDebugGraphic();
```
