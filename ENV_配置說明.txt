# 環境變數配置說明

在 backend 文件夾創建 .env 文件，內容如下：

===========================================
AI 模型設定
===========================================

AGENT_MODEL=gemma3:4b

# 或使用更大的模型
# AGENT_MODEL=gemma3:27b

===========================================
Ollama 連接設定
===========================================

OLLAMA_BASE_URL=http://127.0.0.1:11434

===========================================
使用步驟
===========================================

1. 在 backend 文件夾創建 .env 文件
2. 複製上面的內容到 .env
3. 安裝 Ollama: https://ollama.com
4. 下載模型: ollama pull gemma3:4b
5. 啟動 Ollama: ollama serve
6. 啟動系統: python start_server.py
