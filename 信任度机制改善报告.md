# 信任度机制改善报告

## 📊 改善概述

本次更新对信任度加减机制进行了全面改善，从简单的关键词检测升级为**智能心理模型**，使信任度变化更加真实、动态和符合人类心理。

## 🎯 改善内容

### 1. **心理惯性系统**（Psychological Inertia）

**原理**：基于认知失调理论，信任度越高/越低，改变越困难。

**实现**：
```python
def _calculate_inertia_multiplier(self, current_trust: int, change: int) -> float:
    if change > 0:
        # 正向变化：信任度越高，增加越难
        if current_trust >= 80:
            return 0.6  # 高信任度，增加困难（-40%）
        elif current_trust >= 60:
            return 0.8  # 中高信任度，稍微困难（-20%）
    else:
        # 负向变化：信任度越高，降低越难（更容易找借口）
        if current_trust >= 80:
            return 0.5  # 高信任度，降低很困难（-50%）
        elif current_trust >= 60:
            return 0.7  # 中高信任度，降低较困难（-30%）
```

**效果**：
- ✅ 防止信任度在极端值时快速反转
- ✅ 模拟"一旦相信就很难说服"的真实心理
- ✅ 增加对话轮次，让博弈更有深度

---

### 2. **策略疲劳效应**（Strategy Fatigue）

**原理**：重复使用同一策略，效果递减。

**实现**：
```python
def _calculate_fatigue_multiplier(self, tactics: List[str], recent_tactics: List[List[str]]) -> float:
    for tactic in tactics:
        count = sum(1 for recent in recent_tactics if tactic in recent)
        if count >= 3:
            return 0.5  # 连续3轮以上，效果减半
        elif count >= 2:
            return 0.7  # 连续2轮，效果-30%
        elif count >= 1:
            return 0.9  # 上一轮用过，效果-10%
```

**效果**：
- ✅ 防止骗徒/专家无脑重复同一策略
- ✅ 鼓励多样化的对话策略
- ✅ 更符合真实诈骗情境（受害者会对重复套路产生免疫）

---

### 3. **情绪状态乘数**（Emotional State Multiplier）

**原理**：不同情绪状态对信任度变化的影响不同。

**实现**：
```python
def _calculate_emotional_multiplier(self, change: int, is_scammer: bool) -> float:
    emotional_state = self.victim_trust.emotional_state
    
    if emotional_state == "anxious":
        return 1.3 if is_scammer and change > 0 else 0.8  # 焦虑时更易被骗
    elif emotional_state == "calm":
        return 0.8 if is_scammer and change > 0 else 1.2  # 冷静时更理性
    elif emotional_state == "suspicious":
        return 0.5 if is_scammer and change > 0 else 1.3  # 怀疑时骗徒影响减半
    elif emotional_state == "panicked":
        return 1.5 if is_scammer and change > 0 else 0.7  # 恐慌时极易被操控
```

**效果**：
- ✅ **焦虑(anxious)**：骗徒影响+30%，专家影响-20%
- ✅ **冷静(calm)**：骗徒影响-20%，专家影响+20%
- ✅ **怀疑(suspicious)**：骗徒影响-50%，专家影响+30%
- ✅ **恐慌(panicked)**：骗徒影响+50%，专家影响-30%

### 4. **Persona差异化**（Persona Multiplier）

**原理**：不同persona对同一策略的反应不同。

**实现**：
```python
def _calculate_persona_multiplier(self, tactic: str) -> float:
    if self.victim_persona == "elderly":
        # 长者：对权威和福利特别敏感
        if tactic == "authority": return 1.5  # +50%
        elif tactic == "benefits": return 1.3  # +30%
        elif tactic == "urgency": return 1.2  # +20%
    
    elif self.victim_persona == "average":
        # 普通人：对专业包装和证据敏感
        if tactic == "authority": return 1.1  # +10%
        elif tactic == "evidence": return 1.3  # +30%
    
    elif self.victim_persona == "overconfident":
        # 过度自信：对激将法敏感，对权威不敏感
        if tactic == "authority": return 0.5  # -50%
        elif tactic == "challenge": return 1.5  # +50%
        elif tactic == "urgency": return 0.8  # -20%
```

**效果**：
- ✅ **elderly**：权威策略x1.5，福利策略x1.3，紧急策略x1.2
- ✅ **average**：权威x1.1，证据x1.3
- ✅ **overconfident**：权威x0.5（不吃权威那套），激将法x1.5

### 5. **组合策略加成**（Combination Bonus）

**原理**：多个策略同时使用有协同效果。

**实现**：
```python
def _calculate_combination_bonus(self, tactics: List[str]) -> int:
    bonus = 0
    
    if "authority" in tactics and "urgency" in tactics:
        bonus += 3  # 权威+紧急：经典组合
    
    if "authority" in tactics and "benefits" in tactics:
        bonus += 2  # 权威+福利：吸引力组合
    
    if "urgency" in tactics and "benefits" in tactics:
        bonus += 2  # 紧急+福利：FOMO组合
    
    if len(tactics) >= 3:
        bonus += 3  # 三重组合：超级有效
```

**效果**：
- ✅ **权威+紧急**：+3分（"警察局要求你立即处理"）
- ✅ **权威+福利**：+2分（"政府发放补贴"）
- ✅ **紧急+福利**：+2分（"限时优惠即将截止"）
- ✅ **三重组合**：+3分（综合性骗局）

## 🎮 对玩家的影响

### 1. **对话更有深度**
- 不再是2-3轮就结束
- 需要5-10轮才能看到明显效果
- 更有"博弈感"

### 2. **策略更重要**
- 不能无脑重复同一策略
- 需要根据受害者情绪状态调整
- 组合策略更有效

### 3. **Persona区别明显**
- **elderly**：更容易被权威和福利说服，需要专家温柔安抚
- **average**：需要证据和逻辑，对专业包装敏感
- **overconfident**：对权威免疫，但容易被激将法激发

### 4. **情绪管理重要**
- 焦虑状态下的受害者更容易被骗
- 专家需要先安抚情绪，再提供证据
- 怀疑状态下骗徒很难扭转局面
