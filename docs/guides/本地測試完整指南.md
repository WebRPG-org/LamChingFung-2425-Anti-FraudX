### 步驟 1: 在 Codespace 設置端口為 Public ⭐ **最重要**

1. 在 VS Code 底部找到「**PORTS**」標籤
2. 找到端口 **8000**
3. 右鍵 → Port Visibility → **Public**
4. 確認顯示 🌐 Public

```bash

```

## 🔍 常見問題

### Q1: 連接失敗，顯示 "Failed to fetch"

**解決方案**：
1. 確認端口 8000 是 **Public**（最常見原因）
2. 確認後端正在運行
3. 確認 Codespace 沒有休眠

### Q2: 看到 CORS 錯誤

**解決方案**：
後端已配置 CORS，不應該有此問題。如果還是有，檢查 `backend/main.py`：
```python
app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)
```

### Q3: API 返回 404

**解決方案**：
確認 URL 正確，特別是端點路徑：
- `/` - 健康檢查
- `/chat` - 聊天 API
- `/api/game/start` - 遊戲 API

### Q4: Codespace URL 變化了

**解決方案**：
Codespace 名稱固定，URL 不會變。除非你：
- 刪除並重新創建 Codespace
- 切換到不同的 Codespace
