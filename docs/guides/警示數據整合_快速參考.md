# 防騙警示整合 Fine-Tuning - 快速參考卡

## 🚀 一鍵執行

```bash
# Windows
整合警示數據訓練.bat

# 或分步執行
python backend/scripts/convert_alerts_to_finetuning.py
python backend/scripts/merge_all_training_data.py
python backend/scripts/run_finetuning.py --model expert
```
