# 圖片內容不符修復說明

## 🔍 問題

AI 回應的內容與圖片不符。

**例子**：
- 圖片：WhatsApp 帳號風險詐騙
- AI 回應：exphone.com 網站詐騙

**原因**：
1. 階段 1（圖片理解）提取的內容不準確
2. 階段 2（專家回應）沒有嚴格使用階段 1 的結果

---

## ✅ 解決方案

### 1. 優化階段 1 提示詞

**之前**：
```
請仔細閱讀圖片，提取以下資訊...
```

**現在**：
```python
extraction_prompt = """你是一個圖片內容提取專家。請仔細閱讀圖片，逐字提取所有文字內容。

請按照以下格式回答（用繁體中文）：

**圖片類型**：[WhatsApp/短訊/網站截圖/等]

**完整文字內容**（逐字引用，一字不漏）：
「[圖片中的所有文字，包括標點符號]」

**可疑元素**：
- 網址：[列出所有網址]
- 電話：[列出所有電話號碼]
- 時間限制：[如「12小時內」]
- 聲稱身份：[如「官方」「系統」「銀行」]

**語言**：[繁體中文/簡體中文/英文]

重要：必須逐字引用圖片中的所有文字，不要遺漏任何內容！"""
```

**改進**：
- ✅ 強調「逐字引用」
- ✅ 要求「一字不漏」
- ✅ 結構化輸出格式
- ✅ 明確列出需要提取的元素

---

### 2. 優化階段 2 提示詞

**之前**：
```python
expert_prompt = f"根據以下圖片內容分析...\n\n**圖片內容**：\n{image_description}"
```

**現在**：
```python
expert_prompt = f"""你是香港防詐騙專家。根據以下從圖片中提取的內容，分析這是否為詐騙。

===== 從圖片中提取的內容 =====
{image_description}
================================

用戶問題：{user_message}

重要：
1. 你必須根據上面提取的圖片內容來分析
2. 不要憑空想像或添加圖片中沒有的內容
3. 逐字引用圖片中的文字來說明問題
4. 按照你的專業格式回應（繁體中文廣東話）

請開始分析："""
```

**改進**：
- ✅ 明確標示「從圖片中提取的內容」
- ✅ 強調「必須根據」提取的內容
- ✅ 禁止「憑空想像」
- ✅ 要求「逐字引用」圖片文字

---

### 3. 添加日誌

```python
extracted_content = response['message']['content']
log.info(f"📝 提取的圖片內容：\n{extracted_content[:200]}...")
```

**用途**：
- 可以在日誌中查看階段 1 提取的內容
- 驗證圖片理解是否正確
- 調試問題
