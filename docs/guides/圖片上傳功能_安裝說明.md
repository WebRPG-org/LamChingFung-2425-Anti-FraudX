# 圖片上傳功能 - 安裝說明

## ⚠️ 重要：安裝依賴

圖片上傳功能需要 `aiofiles` 模組來處理異步文件操作。

### 快速安裝

```bash
pip install aiofiles
```

### 完整安裝（推薦）

如果是首次安裝或更新系統，請安裝所有依賴：

```bash
pip install -r backend/requirements.txt
```

## 🐛 故障排除

### 問題 1: ModuleNotFoundError: No module named 'aiofiles'

**解決方法**：
```bash
pip install aiofiles
```

### 問題 2: 權限錯誤

**解決方法**：
```bash
# Windows（以管理員身份運行）
pip install --user aiofiles

# Linux/Mac
sudo pip install aiofiles
```

### 問題 3: 多個 Python 環境

**解決方法**：
```bash
# 使用特定 Python 版本
python3.13 -m pip install aiofiles

# 或使用虛擬環境
python -m venv venv
venv\Scripts\activate  # Windows
source venv/bin/activate  # Linux/Mac
pip install aiofiles
```
