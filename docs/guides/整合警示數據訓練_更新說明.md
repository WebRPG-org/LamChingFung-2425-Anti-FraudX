# 整合警示數據訓練.bat - 更新說明

## 📋 更新內容

### 🎯 新增功能

原本的 `整合警示數據訓練.bat` 只訓練**專家模型**，現在已升級為**專家 + 騙徒雙模型訓練系統**。

### ✨ 主要改進

#### 1. 新增選單系統
```
請選擇訓練模式：
[1] 完整訓練（專家 + 騙徒）- 推薦
[2] 僅訓練專家模型
[3] 僅訓練騙徒模型
[4] 僅生成訓練數據（不訓練）
[0] 退出
```

#### 2. 整合騙徒戰術訓練
- 自動從 `scraped_alerts.json` 提取詐騙手法
- 按 8 大騙局類型生成訓練數據
- 訓練騙徒模型學習真實案例中的詐騙戰術

#### 3. 完整的訓練流程

**選項 [1] - 完整訓練（6 步驟）**：
1. 轉換防騙警示為專家訓練數據
2. 生成騙徒戰術訓練數據（按騙局類型）
3. 生成熱線訓練樣本
4. 合併所有訓練數據
5. 訓練專家模型
6. 訓練騙徒模型

**選項 [2] - 僅訓練專家（4 步驟）**：
1. 轉換防騙警示為專家訓練數據
2. 生成熱線訓練樣本
3. 合併所有訓練數據
4. 訓練專家模型

**選項 [3] - 僅訓練騙徒（2 步驟）**：
1. 生成騙徒戰術訓練數據
2. 訓練騙徒模型

**選項 [4] - 僅生成數據（3 步驟）**：
1. 轉換防騙警示為專家訓練數據
2. 生成騙徒戰術訓練數據
3. 生成熱線訓練樣本

## 🎭 騙徒模型學習內容

### 8 大騙局類型

| 騙局類型 | 身份 | 核心手法 |
|---------|------|---------|
| 假冒官員 | 執法人員 | 權威施壓、製造恐慌、轉駁內地公安 |
| 假冒銀行 | 銀行職員 | 假網站連結、索取密碼、製造帳戶危機 |
| 虛假投資 | 投資顧問 | 高回報承諾、先給甜頭、內部消息 |
| 釣魚詐騙 | 官方機構 | 假連結、假網站、索取個人資料 |
| 中獎詐騙 | 活動主辦 | 製造貪念、限時優惠、要求先繳費 |
| 求職詐騙 | 僱主 | 高薪誘惑、在家工作、收取費用 |
| WhatsApp | 親友 | 假冒熟人、緊急求助、拒絕通話 |
| 愛情詐騙 | 網上情人 | 建立感情、長期培養、製造困境 |

### 4 階段話術訓練

每種騙局類型都包含：
1. **開場話術** - 建立聯繫
2. **施壓話術** - 製造壓力
3. **索取話術** - 獲取資料/金錢
4. **對抗話術** - 反擊專家介入

- 選擇 **[3] 僅訓練騙徒模型**

- 選擇 **[4] 僅生成訓練數據**

## 📊 訓練數據統計

### 專家訓練數據
- **來源**: scraped_alerts.json (270+ 條警示)
- **樣本類型**: 
  - 識別詐騙手法
  - 提供防範建議
  - 案例分析
  - 快速識別技巧
- **預期樣本數**: ~1,080 個（每條警示 4 個樣本）

### 騙徒訓練數據
- **來源**: scraped_alerts.json (270+ 條警示)
- **樣本類型**:
  - 開場話術（權威/利誘/親切）
  - 施壓話術（時間/恐嚇/損失）
  - 索取話術（資料/金錢/先付費）
  - 對抗話術（攻擊專家/製造對立/加強緊迫）
- **預期樣本數**: ~3,240 個（每條警示 12 個樣本）

### 熱線訓練數據
- **來源**: 香港官方熱線資訊
- **樣本類型**: 熱線查詢、求助、舉報
- **預期樣本數**: 根據 `generate_hotline_training_samples.py` 生成

## 🔧 技術細節

### 新增的腳本
- `backend/scripts/generate_scammer_tactics_from_alerts.py`
  - 從警示中提取騙徒手法
  - 按騙局類型分類
  - 生成 4 階段訓練樣本

### 整合的功能
- 自動識別騙局類型（關鍵詞匹配）
- 生成類型專屬的系統提示
- 包含真實案例作為參考
- 生成 JSONL 格式訓練數據

### 訓練參數（推薦）
```
temperature: 0.8
top_p: 0.9
top_k: 40
repeat_penalty: 1.1
```

## 🆚 與其他腳本的關係

### `整合警示數據訓練.bat` vs `騙徒戰術訓練.bat`

| 特點 | 整合警示數據訓練.bat | 騙徒戰術訓練.bat |
|------|---------------------|-----------------|
| **訓練對象** | 專家 + 騙徒（可選） | 僅騙徒 |
| **功能** | 完整訓練流程 | 專注騙徒訓練 |
| **數據生成** | 專家 + 騙徒 + 熱線 | 僅騙徒戰術 |
| **推薦用途** | 首次訓練、完整更新 | 單獨更新騙徒模型 |

**建議**：
- 首次使用或完整更新 → 使用 `整合警示數據訓練.bat` 選擇 [1]
- 只想更新騙徒模型 → 使用 `騙徒戰術訓練.bat` 或 `整合警示數據訓練.bat` 選擇 [3]

## 📞 常見問題

**Q: 訓練需要多長時間？**  
A: 完整訓練（專家 + 騙徒）約 20-40 分鐘，視數據量和硬件而定。

**Q: 可以只訓練騙徒嗎？**  
A: 可以，選擇選項 [3] 僅訓練騙徒模型。

**Q: 訓練數據從哪裡來？**  
A: 從 `data/scraped_alerts.json` 中的真實防騙警示提取。

**Q: 如何驗證訓練效果？**  
A: 啟動模擬系統，測試不同騙局類型，觀察騙徒是否使用對應手法。

**Q: 需要多久訓練一次？**  
A: 當 `scraped_alerts.json` 更新時重新訓練。
