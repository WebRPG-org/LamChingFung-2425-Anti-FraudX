## 📋 4種模式

### 1. 🎮 RPG遊戲模式（當前）

**點擊效果**：關閉選單，開始玩RPG遊戲

**特點**：
- 互動式RPG遊戲
- 與NPC對話學習防詐技巧
- 支持手動/自動訓練
- 人設和騙局輪換

---

### 2. 🤖 自動模擬模式

**點擊效果**：跳轉到自動模擬頁面

**特點**：
- 觀看AI三方對話
- 騙徒 vs 受害者 vs 專家
- 全自動演示詐騙過程
- 選擇人設和騙局類型

**跳轉到**：`/RPG_Project/test_ai_agent_v2.5.html`

---

### 3. 🔧 API測試模式

**點擊效果**：跳轉到API測試頁面

**特點**：
- 測試系統功能
- 檢查API連接
- 開發者工具
- 測試AI回應生成

**跳轉到**：`/RPG_Project/test_plugin.html`

---

### 4. 💬 個人對話模式

**點擊效果**：跳轉到個人對話頁面

**特點**：
- 一對一AI對話
- 防詐助手 / 騙徒模擬
- 互動式學習
- 語音播放功能

**跳轉到**：`/personal_chat.html`

---

## 🎯 如何使用

### 方法1：啟動時選擇

1. **打開遊戲**
   ```
   http://localhost:8000/RPG_Project/rpg_game.html
   ```

2. **看到模式選擇菜單**
   - 自動彈出，覆蓋整個畫面

3. **點擊想要的模式**
   - RPG遊戲模式：關閉菜單，開始遊戲
   - 其他模式：自動跳轉到對應頁面

---

### 方法2：遊戲中切換 ⌨️

**快捷鍵：按 `ESC` 鍵**

1. 在RPG遊戲中，按 `ESC` 鍵
2. 模式選擇菜單重新顯示
3. 選擇想要的模式
4. 切換或繼續遊戲

---

### 方法3：點擊浮動按鈕 🖱️

**右上角浮動按鈕**

1. 關閉模式菜單後
2. 右上角會顯示「📋 切換模式」按鈕
3. 點擊按鈕重新打開菜單
4. 選擇模式

## ⚙️ 技術細節

### 自動檢測環境

```javascript
function switchMode(url) {
    if (window.location.protocol === 'http:' || 
        window.location.protocol === 'https:') {
        // HTTP環境：正常跳轉
        window.location.href = url;
    } else {
        // file://環境：提示錯誤
        alert('請通過HTTP服務器訪問...');
    }
}
```

### 快捷鍵綁定

```javascript
document.addEventListener('keydown', function(e) {

});
```

## 🔧 進階設置

### 如果不想顯示初始菜單

修改 `rpg_game.html`：

```javascript
// 在頁面載入時自動隱藏菜單
window.addEventListener('load', function() {
    closeModeSelect();
});
```

### 如果想添加新模式

在 `mode-grid` 中添加新的卡片：

```html
<div class="mode-option" onclick="switchMode('/your-new-mode.html')">
    <div class="mode-option-icon">🆕</div>
    <div class="mode-option-title">新模式</div>
    <div class="mode-option-desc">
        模式說明
    </div>
</div>
```
