# 圖片上傳功能 - 快速開始

## 🚀 一分鐘快速開始

### 1. 啟動系統
```batch
一键启动.bat
```

### 2. 打開瀏覽器
訪問：http://localhost:8000

### 3. 開始使用
1. 點擊「個人對話模式」
2. 選擇「防詐助手」
3. 點擊 📷 按鈕上傳圖片
4. 發送並查看 AI 分析

---

## 📸 功能亮點

| 功能 | 說明 |
|------|------|
| **多圖上傳** | 一次上傳多張圖片 |
| **實時預覽** | 上傳後即時顯示縮略圖 |
| **燈箱查看** | 點擊圖片全屏查看 |
| **AI 分析** | 自動分析圖片內容 |
| **移除功能** | 可單獨移除不需要的圖片 |

## 📊 技術規格

### 支持的格式
- ✅ JPG / JPEG
- ✅ PNG
- ✅ GIF
- ✅ WebP
- ✅ BMP
- ✅ SVG

### 限制
- **圖片大小**: 建議 < 5MB
- **圖片數量**: 無限制（建議 ≤ 5 張）
- **存儲位置**: `backend/uploads/chat_images/`

## 🐛 故障排除

### 問題 1: 圖片上傳沒反應
**解決方法**：
1. 檢查後端服務是否運行
2. 查看瀏覽器控制台（F12）
3. 確認圖片格式支持

### 問題 2: AI 無法分析圖片
**解決方法**：
1. 確認圖片已成功上傳
2. 檢查後端日誌
3. 嘗試重新發送

### 問題 3: 預覽顯示異常
**解決方法**：
1. 清除瀏覽器緩存
2. 重新整理頁面
3. 嘗試其他瀏覽器
