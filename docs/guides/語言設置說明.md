# 語言設置說明

## ✅ 已修復

### 問題
AI 回應使用英文，而不是繁體中文廣東話。

### 解決方案
在 `vision_service.py` 中添加了**語言要求**（最高優先級）：

```python
language_instruction = """
🌐 **語言要求（最高優先級）**：
- ✅ **必須使用繁體中文**
- ✅ **必須使用香港廣東話口語**
- ✅ **禁止使用英文回應**（除非引用原文）
- ✅ **禁止使用簡體中文**
- ✅ **禁止使用普通話用語**

例如：
- ✅ 正確：「呢個係詐騙短訊」「唔好點擊」「立即停止」
- ❌ 錯誤：「这是诈骗」「不要点击」「This is a scam」

**無論圖片中的文字是什麼語言，你的回應都必須用繁體中文廣東話！**
"""
```

## 💡 語言要求的優先級

### 系統提示詞結構

```
1. 語言要求（最高優先級）
   ↓
2. ExpertAgent 的專業指引
   ↓
3. 圖片分析指引
   ↓
4. 用戶消息
```

### 為什麼這樣設計？

**問題**：圖片中的文字是簡體中文，AI 可能自動匹配語言。

**解決**：在最前面明確要求使用繁體中文廣東話，並給出正反例。

## 🔧 技術細節

### 修改的文件

**`backend/services/vision_service.py`**

```python
# 構建完整的系統提示詞
full_system_prompt = language_instruction + "\n\n" + system_prompt

messages = [
    {
        'role': 'system',
        'content': full_system_prompt  # 語言要求 + 專業指引
    },
    {
        'role': 'user',
        'content': message,
        'images': images
    }
]
```

### 語言要求的關鍵點

1. **明確禁止** - 不只說「使用廣東話」，還要說「禁止英文」
2. **給出例子** - 正確和錯誤的對比
3. **強調優先級** - 「最高優先級」「無論圖片是什麼語言」
4. **具體用詞** - 列出廣東話特有詞彙（「呢個」「唔好」「係」）
