# 個人對話模式 - 語音功能說明

## 📋 新增功能概覽

本次更新為個人對話模式添加了完整的語音交互功能，包括語音輸入和自動語音播放。

---

## ✨ 新增功能

### 1. 🏠 返回主頁按鈕（左上角）
- **位置**：頁面頭部左上角
- **樣式**：半透明白色按鈕，hover效果
- **功能**：一鍵返回系統主頁
- **快捷鍵**：暫無（可擴展）

### 2. 🎤 語音輸入功能
- **位置**：聊天輸入框左側
- **圖標**：麥克風 🎤（錄音時變為紅色 🔴）
- **功能**：
  - 點擊麥克風開始語音識別
  - 支持廣東話（zh-HK）
  - 識別結果自動填入輸入框
  - 再次點擊或自動停止錄音
- **動畫效果**：錄音時紅色脈衝動畫
- **瀏覽器支持**：Chrome、Edge、Safari

### 3. 🔊 自動語音播放功能
- **位置**：聊天容器頂部設置面板
- **開關**：複選框（默認啟用）
- **功能**：
  - AI回復自動轉語音播放
  - 支持廣東話語音合成
  - 可隨時開啟/關閉
  - 歡迎消息也會自動播放
- **語音參數**：
  - 語言：廣東話（zh-HK）
  - 語速：1.0（正常）
  - 音調：1.0（正常）
  - 音量：1.0（最大）

## 🔧 技術實現

### 1. Web Speech API

#### 語音識別（Speech Recognition）
```javascript
const SpeechRecognition = window.SpeechRecognition || window.webkitSpeechRecognition;
recognition = new SpeechRecognition();
recognition.lang = 'zh-HK'; // 廣東話
recognition.continuous = false;
recognition.interimResults = false;
```

**支持的瀏覽器**：
- ✅ Chrome / Edge (推薦)
- ✅ Safari
- ❌ Firefox (不支持)

#### 語音合成（Speech Synthesis）
```javascript
const utterance = new SpeechSynthesisUtterance(text);
utterance.lang = 'zh-HK'; // 廣東話
utterance.rate = 1.0; // 語速
utterance.pitch = 1.0; // 音調
utterance.volume = 1.0; // 音量
speechSynthesis.speak(utterance);
```

**支持的瀏覽器**：
- ✅ Chrome / Edge (推薦)
- ✅ Safari
- ✅ Firefox

## 🚀 使用指南

### 語音輸入步驟

1. **開始對話**
   - 選擇對話模式（防詐助手/騙徒模擬）
   - 點擊「開始對話」

2. **使用語音輸入**
   - 點擊輸入框左側的麥克風圖標 🎤
   - 允許瀏覽器使用麥克風權限（首次使用）
   - 開始說話（麥克風變紅色 🔴）
   - 說完後自動停止，文字填入輸入框
   - 檢查文字，點擊「發送」

3. **快捷操作**
   - 直接點擊發送：使用鍵盤輸入
   - 再次點擊紅點：手動停止錄音
   - Enter鍵：發送消息

### 語音播放設置

1. **默認設置**
   - 自動播放：✅ 啟用
   - 可在對話開始後關閉

2. **調整設置**
   - 點擊「🔊 自動播放語音」複選框
   - ✅ 勾選：AI回復自動播放語音
   - ☐ 不勾選：靜音模式，不播放語音

3. **播放特點**
   - 每次只播放一段語音
   - 新語音會停止舊語音
   - 關閉自動播放不影響語音輸入

## ⚠️ 注意事項

### 語音識別

1. **麥克風權限**
   - 首次使用需允許瀏覽器訪問麥克風
   - 如被拒絕，需在瀏覽器設置中手動允許

2. **網絡連接**
   - 語音識別需要網絡連接
   - Chrome使用Google的語音API

3. **識別準確度**
   - 清晰的發音效果更好
   - 環境噪音會影響識別
   - 廣東話識別取決於語音引擎

4. **不支持的瀏覽器**
   - Firefox不支持Web Speech Recognition API
   - 會提示切換到Chrome/Edge/Safari

### 語音合成

1. **語音引擎**
   - 依賴系統安裝的語音包
   - Windows 10/11有內置廣東話語音
   - macOS需額外下載廣東話語音

2. **自動播放限制**
   - 某些瀏覽器限制自動播放音頻
   - 可能需要用戶先互動（點擊）

3. **多語言支持**
   - 優先使用廣東話語音
   - 如無廣東話，使用系統默認中文
   - 可在控制台查看可用語音

## 🐛 故障排除

### 語音識別問題

#### 問題：點擊麥克風沒反應
**解決方案**：
1. 檢查瀏覽器是否支持（Chrome/Edge/Safari）
2. 檢查麥克風權限是否允許
3. 重新加載頁面
4. 查看控制台錯誤信息

#### 問題：識別結果不準確
**解決方案**：
1. 說話更清晰、語速適中
2. 減少環境噪音
3. 靠近麥克風
4. 確認語言設置為廣東話

### 語音合成問題

#### 問題：沒有聲音
**解決方案**：
1. 檢查「自動播放語音」是否勾選
2. 檢查系統音量和瀏覽器音量
3. 確認沒有其他應用占用音頻
4. 查看控制台是否有錯誤

#### 問題：語音不是廣東話
**解決方案**：
1. 安裝系統廣東話語音包
   - Windows：設定→時間與語言→語音→管理語音
   - macOS：系統偏好設定→輔助使用→語音內容
2. 重啟瀏覽器
3. 檢查控制台可用語音列表
