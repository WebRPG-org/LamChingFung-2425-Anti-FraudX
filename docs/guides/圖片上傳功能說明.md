# 圖片上傳功能說明

## 📸 功能概述

Web UI 現已支持**圖片上傳功能**，用戶可以在對話中上傳圖片，讓 AI 分析圖片內容並判斷是否為詐騙。

---

## ✨ 主要特點

### 1. 多圖片上傳
- ✅ 支持一次上傳多張圖片
- ✅ 支持常見圖片格式（JPG, PNG, GIF, WebP 等）
- ✅ 實時預覽上傳的圖片

### 2. 圖片預覽
- ✅ 上傳後即時顯示縮略圖
- ✅ 可單獨移除不需要的圖片
- ✅ 點擊圖片可全屏查看（燈箱效果）

### 3. AI 分析
- ✅ **防詐助手模式**：分析圖片內容，判斷是否為詐騙
- ✅ **騙徒模式**：根據圖片內容調整詐騙策略
- ✅ 圖片自動保存到服務器

---

## 🎯 使用方法

### 步驟 1: 啟動系統

```batch
一键启动.bat
```

或手動啟動：
```bash
python start_server.py
```

### 步驟 2: 打開 Web UI

訪問：http://localhost:8000

### 步驟 3: 選擇對話模式

1. 點擊「個人對話模式」
2. 選擇「防詐助手」或「騙徒模擬」
3. 點擊「開始對話」

### 步驟 4: 上傳圖片

在聊天界面底部：

1. **點擊 📷 按鈕** - 選擇圖片文件
2. **預覽圖片** - 確認上傳的圖片
3. **輸入文字**（可選）- 描述或提問
4. **點擊發送** - 發送圖片和文字

## 🔧 技術細節

### 前端實現

#### 圖片選擇
```javascript
function handleImageSelect(event) {
    const files = event.target.files;
    for (let file of files) {
        if (file.type.startsWith('image/')) {
            const reader = new FileReader();
            reader.onload = function(e) {
                selectedImages.push({
                    file: file,
                    dataUrl: e.target.result
                });
                updateImagePreview();
            };
            reader.readAsDataURL(file);
        }
    }
}
```

#### 圖片預覽
```javascript
function updateImagePreview() {
    const container = document.getElementById('imagePreviewContainer');
    container.innerHTML = '';
    
    selectedImages.forEach((img, index) => {
        const preview = document.createElement('div');
        preview.className = 'image-preview';
        preview.innerHTML = `
            <img src="${img.dataUrl}" alt="預覽" />
            <button class="remove-image" onclick="removeImage(${index})">×</button>
        `;
        container.appendChild(preview);
    });
}
```

#### 發送圖片
```javascript
async function sendMessage() {
    const formData = new FormData();
    formData.append('session_id', currentSessionId);
    formData.append('message', message || '');
    
    selectedImages.forEach((img) => {
        formData.append('images', img.file);
    });
    
    const response = await fetch(`${API_BASE_URL}/api/personal-chat/message`, {
        method: 'POST',
        body: formData
    });
}
```

## ⚙️ 配置選項

### 支持的圖片格式
```html
<input type="file" accept="image/*" multiple />
```

可修改為特定格式：
```html
<input type="file" accept="image/jpeg,image/png,image/gif" multiple />
```

### 圖片大小限制

在後端添加大小檢查：
```python
MAX_FILE_SIZE = 5 * 1024 * 1024  # 5MB

if len(content) > MAX_FILE_SIZE:
    raise HTTPException(400, "圖片太大，請上傳小於 5MB 的圖片")
```

### 圖片數量限制

在前端添加數量檢查：
```javascript
const MAX_IMAGES = 5;

if (selectedImages.length >= MAX_IMAGES) {
    alert(`最多只能上傳 ${MAX_IMAGES} 張圖片`);
    return;
}
```

## 🐛 常見問題

### Q1: 圖片上傳後沒有反應？
**A**: 檢查以下項目：
1. 後端服務是否運行
2. 瀏覽器控制台是否有錯誤
3. 圖片格式是否支持
4. 圖片大小是否過大

### Q2: AI 無法分析圖片內容？
**A**: 目前 AI 分析基於圖片文件名和上下文，未來可整合 OCR 或圖像識別 API。

### Q3: 圖片預覽顯示不正常？
**A**: 
1. 清除瀏覽器緩存
2. 檢查圖片格式
3. 嘗試重新上傳

### Q4: 如何刪除已上傳的圖片？
**A**: 
- 前端：點擊預覽圖片上的 × 按鈕
- 服務器：手動刪除 `backend/uploads/chat_images/` 中的文件
