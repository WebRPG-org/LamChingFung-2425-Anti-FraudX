## 🚀 使用方法

### 方法1：一鍵執行（推薦）

```bash
整合警示數據訓練.bat
```

這會自動執行：
1. 轉換防騙警示（包含熱線資訊）
2. 生成熱線訓練樣本
3. 合併所有數據
4. 訓練模型

### 方法2：分步執行

```bash
# 步驟1：轉換警示數據（已包含熱線）
python backend/scripts/convert_alerts_to_finetuning.py

# 步驟2：生成熱線訓練樣本
python backend/scripts/generate_hotline_training_samples.py

# 步驟3：合併所有數據
python backend/scripts/merge_all_training_data.py

# 步驟4：訓練模型
python backend/scripts/run_finetuning.py --model expert
```

## 🔧 進階配置

### 增加熱線訓練權重

如果需要進一步強化熱線記憶，可以：

#### 1. 增加熱線樣本數量

編輯 `generate_hotline_training_samples.py`，添加更多變體：

```python
# 添加更多詢問方式
samples.append({
    "user": "防騙熱線係咩號碼？",
    "assistant": "防騙易熱線係 18222..."
})

samples.append({
    "user": "我想查證詐騙，打咩電話？",
    "assistant": "打防騙易熱線 18222..."
})
```

#### 2. 重複熱線樣本

在合併腳本中重複加載熱線樣本：

```python
# 加載熱線樣本3次（增加權重）
for i in range(3):
    hotline_samples = self.load_jsonl_files("hotline_training_*.jsonl")
    all_samples.extend(hotline_samples)
```

#### 3. 調整訓練參數

在 Modelfile 中：

```
PARAMETER repeat_penalty 1.0  # 降低重複懲罰
PARAMETER temperature 0.6     # 降低隨機性
```
