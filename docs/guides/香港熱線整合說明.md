## 📞 熱線列表

### 1. **18222** (防騙易熱線)
- **用途**：24小時反詐騙諮詢熱線
- **何時用**：已經被騙、懷疑被騙、需要諮詢

### 2. **999** (緊急求助)
- **用途**：警察、消防、救護
- **何時用**：緊急情況、需要立即報警

### 3. **2860 5012** (反詐騙協調中心)
- **用途**：舉報詐騙
- **何時用**：想舉報詐騙案件

### 4. **3423 6611** (個人資料防騙熱線)
- **用途**：私隱專員公署，懷疑個人資料被誘騙
- **何時用**：個人資料被盜、被誘騙提供資料

### 5. **2929 2222** (消費者委員會)
- **用途**：消費投訴
- **何時用**：消費糾紛、產品問題

## 🎯 何時提供熱線

### ✅ 應該提供的情況

1. **用戶已經被騙**
   ```
   如果你已經俾咗錢或資料，立即打：
   📞 18222 (防騙易熱線) - 24小時服務
   📞 999 (報警) - 緊急情況
   ```

2. **用戶想舉報詐騙**
   ```
   你可以打去舉報：
   📞 2860 5012 (反詐騙協調中心)
   ```

3. **用戶個人資料被盜**
   ```
   如果你已經俾咗個人資料：
   📞 3423 6611 (個人資料防騙熱線)
   ```

4. **用戶問「應該點做」**
   ```
   有懷疑可以打去問：
   📞 18222 (防騙易熱線)
   ```

### ❌ 不應該提供的情況

1. **只係分析圖片**
   - 用戶只係想知道「呢個係咪詐騙」
   - 未發生任何損失
   - 只係預防性質

2. **用戶未問**
   - 唔好主動提供太多熱線
   - 避免資訊過載

3. **只係預防性質**
   - 用戶只係學習防騙知識
   - 未遇到實際詐騙

## 📊 提供熱線的原則

### 原則 1: 按需提供

**唔好**：
```
❌ 每次都列出所有 5 個熱線
❌ 用戶未問就主動提供
```

**應該**：
```
✅ 根據情況提供相關熱線
✅ 用戶問「點做」才提供
```

### 原則 2: 簡單格式

**唔好**：
```
❌ 你可以致電香港警務處反詐騙協調中心，電話號碼為 2860 5012，辦公時間為...
```

**應該**：
```
✅ 📞 2860 5012 (反詐騙協調中心)
```

### 原則 3: 優先級

**高風險情況（已被騙）**：
1. 18222 (防騙易熱線)
2. 999 (報警)

**一般情況（想舉報）**：
1. 2860 5012 (反詐騙協調中心)

**資料被盜**：
1. 3423 6611 (個人資料防騙熱線)

## 💡 設計考量

### 1. 不主動提供

**原因**：
- 避免資訊過載（特別是老人家）
- 只在需要時提供
- 保持回應簡潔

### 2. 按需提供

**原因**：
- 根據情況提供相關熱線
- 不是每次都列出所有 5 個
- 更有針對性

### 3. 簡單格式

**原因**：
- 老人家容易記住
- 清楚標示用途
- 容易撥打
