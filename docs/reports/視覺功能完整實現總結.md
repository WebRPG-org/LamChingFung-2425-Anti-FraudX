## 🔧 核心文件

### 1. `backend/services/vision_service.py`

**功能**：混合模式視覺分析服務

**關鍵方法**：
- `analyze_with_vision()` - 主入口，協調兩階段
- `_extract_image_content()` - 階段 1：提取圖片內容
- `_generate_expert_response()` - 階段 2：生成專家回應

**模型配置**：
```python
self.vision_model = os.getenv("VISION_MODEL", "gemma3:4b")
self.expert_model = os.getenv("AGENT_MODEL_EXPERT", "anti-fraud-expert-20251117_173453")
```

---

### 2. `backend/agents/expert.py`

**功能**：ExpertAgent 的專業指引

**關鍵內容**：
- 圖片分析專門指引
- 結構化輸出格式
- 老人家友善原則
- 香港重要熱線

**格式要求**：
```
🚨 呢個係詐騙！唔好點擊！

⚠️ 點解係詐騙：[3 個重點]

```

---

### 3. `backend/api/personal_chat_routes.py`

**功能**：處理圖片上傳和分析請求

**關鍵邏輯**：
```python
if image_base64_list:
    # 有圖片 → 使用 VisionService
    from agents.expert import ExpertAgent
    expert_agent = ExpertAgent()
    system_prompt = expert_agent.instruction
    
    reply = await vision_service.analyze_with_vision(
        message=context_message,
        images=image_base64_list,
        system_prompt=system_prompt
    )
else:
    # 無圖片 → 使用 AgentService
    response = await agent_service.generate_response(...)
```

## 🎯 關鍵優化

### 1. 階段 1：準確提取

**提示詞**：
```
你是一個圖片內容提取專家。請仔細閱讀圖片，逐字提取所有文字內容。

**完整文字內容**（逐字引用，一字不漏）：
「[圖片中的所有文字，包括標點符號]」

重要：必須逐字引用圖片中的所有文字，不要遺漏任何內容！
```

**關鍵詞**：
- 「逐字引用」
- 「一字不漏」
- 「不要遺漏」

---

### 2. 階段 2：嚴格遵守

**提示詞**：
```
===== 從圖片中提取的內容 =====
{image_description}
================================

**必須使用以下格式回應**：
🚨 [係詐騙 / 唔係詐騙]！
...

重要規則：
- 必須根據上面提取的圖片內容
- 不要憑空想像或添加內容
- 逐字引用圖片中的文字
```

**關鍵詞**：
- 「必須根據」
- 「不要憑空想像」
- 「逐字引用」

---

### 3. 格式控制

**要求**：
- 只用 5 個 emoji：🚨📱📝⚠️💡
- 只講 3 個重點
- 每點最多 15 字
- 繁體中文廣東話

---

### 4. 老人家友善

**原則**：
- 開門見山（第一句講結論）
- 用簡單字（避免專業術語）
- 只講 3 個重點（不要太多資訊）
- 行動具體（「刪除」「封鎖」）

## 🔄 工作流程

### 完整流程

1. **用戶上傳圖片**
   - 前端：`personal_chat.html`
   - 轉換為 base64

2. **後端接收**
   - API：`personal_chat_routes.py`
   - 保存圖片到 `uploads/chat_images/`

3. **階段 1：圖片理解**
   - 模型：`gemma3:4b`
   - 任務：逐字提取文字內容
   - 輸出：結構化描述

4. **階段 2：專家分析**
   - 模型：`anti-fraud-expert`
   - 輸入：階段 1 的結果 + 格式要求
   - 輸出：結構化建議

5. **返回用戶**
   - 格式化的防詐騙建議
   - 繁體中文廣東話
