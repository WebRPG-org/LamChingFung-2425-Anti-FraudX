## 🎯 功能特點

### 1. 智能數據轉換

- ✅ 自動解析警示內容（手法 + 呼籲）
- ✅ 每條警示生成 4 種訓練場景
- ✅ 保留元數據（來源、日期、連結）
- ✅ 自動分割訓練集和驗證集（90/10）

### 2. 多場景訓練樣本

每條警示自動生成：

1. **識別詐騙手法**：解釋騙局類型和特徵
2. **提供防範建議**：給出具體行動步驟
3. **案例分析**：引用真實案例說明
4. **快速識別技巧**：教授識別方法

### 3. 數據整合

- ✅ 合併對話訓練數據
- ✅ 合併警示訓練數據
- ✅ 自動生成 Modelfile
- ✅ 統計信息記錄

```bash
python backend/scripts/convert_alerts_to_finetuning.py

# 直接訓練
python backend/scripts/run_finetuning.py --model expert --data-pattern "alerts_expert_*.jsonl"
```

## 🔧 技術細節

### 數據格式

#### 輸入格式（scraped_alerts.json）

```json
{
  "title": "提防假冒警務人員電話騙案",
  "date": "2021-10-08",
  "link": "https://www.adcc.gov.hk/...",
  "source": "ADCC",
  "content": "手法\n...\n警方呼籲\n..."
}
```

#### 輸出格式（JSONL）

```json
{
  "messages": [
    {"role": "system", "content": "你是香港反詐騙專家黃sir..."},
    {"role": "user", "content": "黃sir，我想了解..."},
    {"role": "assistant", "content": "好，等我同你講解下..."}
  ],
  "metadata": {
    "source": "scraped_alerts",
    "alert_id": "abc123",
    "title": "...",
    "date": "2021-10-08",
    "type": "scam_identification"
  }
}
```

### 轉換邏輯

```python
# 每條警示 → 4 個訓練樣本
for alert in alerts:
    samples = [
        generate_identification_sample(alert),   # 識別手法
        generate_prevention_sample(alert),       # 防範建議
        generate_case_analysis_sample(alert),    # 案例分析
        generate_quick_tips_sample(alert)        # 快速識別
    ]
```
