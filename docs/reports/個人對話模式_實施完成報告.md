## 🔧 技術實現

### 架構設計

```
前端 (personal_chat.html)
    ↓ HTTP/JSON
後端 API (personal_chat_routes.py)
    ↓
AgentService
    ↓
Expert Agent / Scammer Agent
    ↓
Ollama LLM
```

### 關鍵技術

**後端**：
- FastAPI 異步處理
- Pydantic 數據驗證
- Google ADK Agent 系統
- 內存會話管理

**前端**：
- 純 JavaScript（無需框架）
- Fetch API 非同步通信
- CSS3 動畫和過渡
- LocalStorage 暫存（未來可擴展）

**AI 模型**：
- 助手：`anti-fraud-expert-20251112_095003`
- 騙徒：`anti-fraud-scammer-20251112_095003`
- 基礎：`gemma3:4b`

### 數據流

```json
// 1. 開始會話
POST /api/personal-chat/start
{
    "mode": "assistant",
    "scam_type": null
}
→ 返回 session_id 和歡迎消息

// 2. 發送消息
POST /api/personal-chat/message
{
    "session_id": "uuid",
    "message": "用戶輸入"
}
→ 返回 AI 回應和更新的歷史

// 3. 對話歷史格式
{
    "history": [
        {"role": "user", "content": "..."},
        {"role": "assistant", "content": "..."}
    ]
}
```

## 🚀 快速開始

### 步驟1：啟動後端

```bash
cd backend
python main.py
```

### 步驟2：打開界面

瀏覽器訪問：
```
http://localhost:8000/personal_chat.html
```

### 步驟3：選擇模式

- 點擊「防詐助手」或「騙徒模擬」
- 如選擇騙徒模擬，選擇騙局類型
- 點擊「開始對話」

### 步驟4：開始對話

- 輸入您的消息
- 按 Enter 或點擊「發送」
- 查看 AI 回應
