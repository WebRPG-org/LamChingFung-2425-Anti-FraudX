## 📋 項目概覽

AI 防詐平台 v4.1 是一個完整的防詐教育和分析系統，通過四個智能 Agent 的協作，模擬真實詐騙場景，幫助用戶學習防騙知識。

### 核心特性

1. **四代理系統** - 騙徒、專家、受害者、記錄員
2. **動態信任度系統** - 實時追蹤信任度變化
3. **多維度性能評分** - 騙徒和專家的詳細評分
4. **高效並行生成** - 性能提升 50-70%
5. **完整的會話管理** - 創建、恢復、分析
6. **多格式數據導出** - JSON、CSV、PDF、Excel

## 🎯 核心功能

### 1. 四代理系統

**ScammerAgent** - 模擬真實詐騙
- 策略漸進性：自動進入不同的策略階段
- 人格適應：根據受害者類型調整話術
- 字數限制：50-100 字，保持自然

**ExpertAgent** - 防詐專家
- 人格策略：針對不同人格的介入方式
- 具體建議：提供可執行的防騙步驟
- 官方熱線：集成香港官方防騙熱線

**VictimAgent** - 受害者模擬
- 情緒狀態：5 種情緒狀態的動態變化
- 初始信任度：根據人格設置初始值
- 自然反應：根據情緒生成自然的受害者反應

**RecorderAgent** - 分析記錄員
- 結果判定：SUCCESS、FAILURE、PARTIAL
- 性能評分：多維度的深入分析
- 改進建議：針對性的改進方向

### 2. 信任度系統

**動態追蹤**
- 初始信任度：根據人格設置
- 實時更新：根據消息內容更新
- 軌跡分析：記錄信任度變化過程

**信任度修改器**
- 騙徒話術：+5 到 +10
- 專家介入：-5 到 -10
- 情緒狀態：±5 到 ±10

### 3. 性能評分系統

**騙徒評分**（6 維度）
- persuasiveness（說服力）× 0.30
- credibility（可信度）× 0.25
- pressure_effectiveness（施壓效果）× 0.25
- strategy_consistency（策略一致性）× 0.20

**專家評分**（5 維度）
- intervention_effectiveness（干預效果）× 0.30
- clarity（清晰度）× 0.20
- empathy（同理心）× 0.20
- actionability（可執行性）× 0.15
- timing（時機把握）× 0.15

### 4. 並行生成系統

**三種模式**
- full：生成騙徒、專家、受害者三個回應
- expert_only：只生成專家回應
- scammer_only：只生成騙徒回應

**性能優化**
- asyncio.gather() 並行執行
- 執行時間追蹤
- 異常處理和錯誤恢復

## 🚀 使用示例

### 基本使用

```python
from services.agent_service import AgentService

# 初始化服務
service = AgentService(persona_type="elderly", scam_type="banking")
session_id = service.create_session()

# 生成回應
response = await service.generate_response(
    agent_type="scammer",
    message="你好，我係銀行職員",
    session_id=session_id
)

print(response["reply"])
print(f"對騙徒信任度: {response['trust_in_scammer']}")
```

### 並行生成

```python
# 並行生成三個回應
parallel = await service.generate_parallel_responses(
    victim_message="我唔知點算好",
    session_id=session_id,
    mode="full"
)

print(f"執行時間: {parallel['execution_time_ms']}ms")
```

### 最終分析

```python
# 生成最終分析
history = service.get_session_history(session_id)
analysis = await service.generate_final_analysis(history)

print(f"騙徒評分: {analysis['scammer_performance']['overall_score']}")
print(f"專家評分: {analysis['expert_performance']['overall_score']}")
```
