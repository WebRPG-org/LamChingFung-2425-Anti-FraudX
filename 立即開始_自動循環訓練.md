# 🚀 立即開始：自動循環訓練

## ✅ 所有功能已完成！

您的 RPG Maker 訓練系統現在已完全升級，具備：
- ✅ 實時對話顯示
- ✅ 自動循環訓練（完成6個NPC後自動返回第1個）
- ✅ 詳細統計追蹤
- ✅ 自動數據保存

## 🎮 三步驟開始訓練

### 步驟 1：確認插件配置

打開 RPG Maker MV，檢查 Plugin Manager：

**必須啟用：**
- ✅ `SimulationTraining_Compatible` - **ON**

**建議禁用（避免衝突）：**
- ❌ `RotatingScamSystem` - **OFF**

### 步驟 2：啟動後台服務

打開終端，執行：
```bash
cd C:\Users\fungr\Documents\AgentV3\AI-Agent
python backend/main.py
```

等待看到：
```
INFO:     Application startup complete.
INFO:     Uvicorn running on http://0.0.0.0:8000
```

### 步驟 3：開始訓練

在 RPG Maker MV 中：
1. 按 **F5** 進入測試模式
2. 按 **F8** 打開開發者控制台
3. 在控制台輸入並執行：
   ```javascript
   StartAutoMode
   ```

## 🛑 停止訓練

在控制台執行：
```javascript
StopAutoMode
```

系統會：
- 停止自動移動
- 關閉當前連接
- 恢復玩家控制
- 保留所有統計數據

## 💡 實用提示

### 長期訓練
如果您想進行長期訓練（例如過夜）：
1. 執行 `StartAutoMode`
2. 確保電腦不會休眠
3. 讓系統自動運行
4. 早上回來查看 `ShowStats`

預期結果（8小時）：
- 約 25-30 輪
- 約 150-180 次對話
- 約 150+ 個 JSON 文件

### 監控訓練
- **RPG Maker 窗口** - 查看對話顯示
- **控制台（F8）** - 查看日誌信息
- **終端窗口** - 查看後台日誌
- **ShowStats** - 查看統計數據

### 故障排除

**問題：對話沒有顯示**
- 檢查後台服務是否運行
- 檢查控制台是否有錯誤
- 嘗試 `StopAutoMode` 後重新 `StartAutoMode`

**問題：訓練在某個NPC停止**
- 檢查 NPC 是否可到達
- 查看控制台錯誤信息
- 重新啟動訓練

**問題：循環沒有自動開始**
- 等待 3 秒
- 檢查 `autoModeActive` 狀態
- 查看控制台日誌
