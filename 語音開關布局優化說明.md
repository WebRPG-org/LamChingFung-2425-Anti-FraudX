## 🎨 設計特點

### 1. **標準化佈局** ✅

遵循主流UI設計規範：
- **左側**：標籤文字（說明功能）
- **右側**：開關控件（操作區域）

這與以下平台一致：
- iOS 設置頁面
- Android 設置頁面
- macOS 系統偏好設置
- Windows 設定應用

---

### 2. **視覺平衡** ✅

```
🔊 自動播放語音 ───────── ━━━🔘
   文字標籤              開關控件
   (固定位置)            (固定位置)
```

**優點**：
- 文字始終在同一位置
- 開關始終在同一位置
- 易於快速掃描
- 操作區域明確

---

### 3. **空間分配** ✅

使用 `justify-content: space-between`：

```css
.voice-settings label {
    display: flex;
    justify-content: space-between;  /* 左右分佈 */
    gap: 15px;
}
```

**效果**：
```
[🔊 自動播放語音]  <----空白----->  [━━━🔘]
      文字                            開關
   (靠左對齊)                      (靠右對齊)
```

## 💡 技術實現

### HTML 結構

**優化前**：
```html
<label>
    <input type="checkbox" id="autoSpeakToggle" checked />
    🔊 自動播放語音
</label>
```

**優化後**：
```html
<label>
    <span class="voice-label">🔊 自動播放語音</span>
    <input type="checkbox" id="autoSpeakToggle" checked />
</label>
```

**改進點**：
- 用 `<span>` 包裹文字標籤
- 調整順序：文字在前，開關在後
- 便於單獨控制文字樣式

---

### CSS 樣式

#### 主容器樣式
```css
.voice-settings label {
    display: flex;
    align-items: center;
    justify-content: space-between;  /* 關鍵：左右分佈 */
    gap: 15px;
}
```

#### 文字標籤樣式
```css
.voice-settings label .voice-label {
    font-size: 15px;
    font-weight: 500;
    color: #333;
    display: flex;
    align-items: center;
    gap: 8px;  /* emoji 和文字的間距 */
}
```

#### 開關樣式
```css
.voice-settings input[type="checkbox"] {
    width: 50px;
    height: 28px;
    flex-shrink: 0;  /* 關鍵：防止被壓縮 */
}
```

## 🔍 細節優化

### 防止壓縮

```css
.voice-settings input[type="checkbox"] {
    flex-shrink: 0;  /* 確保開關不會被壓縮 */
}
```

**作用**：
- 保持開關固定尺寸 50x28px
- 即使文字很長也不會擠壓開關
- 保證觸控區域大小

---

### 文字換行處理

```css
.voice-settings label .voice-label {
    display: flex;
    align-items: center;
    gap: 8px;
}
```

**作用**：
- emoji 和文字保持在同一行
- 如果空間不足，文字會自然換行
- 開關始終在右側固定位置
