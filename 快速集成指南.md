**文件**: `backend/scripts/real_dialogue_runner.py` 或 `backend/api/simulation_routes.py`

**修改位置**: 在構建prompt的地方

**原代碼**:
```python
# 原來直接使用完整對話歷史
prompt = "\n".join([f"{turn['speaker']}：{turn['dialogue']}" for turn in self.conversation_history])
```

**修改為**:
```python
from utils.conversation_summarizer import compress_conversation_history

# 使用壓縮後的對話歷史
prompt = compress_conversation_history(
    conversation_history=self.conversation_history,
    additional_context=f"你是{agent_name}..."  # 可選的額外上下文
)
```


### 第三步：集成動態策略調整（可選但推薦）

**文件**: `backend/scripts/real_dialogue_runner.py`

**修改位置**: 在騙徒回合前

**添加策略管理**:
```python
from utils.scammer_strategy_manager import (
    recommend_scammer_strategy,
    build_strategy_enhanced_prompt
)

class RealDialogueRunner:
    def __init__(self, ...):
        # ... 現有代碼 ...
        self.current_strategy = None  # 新增：追蹤當前策略
        self.last_trust_scammer = 70  # 新增：追蹤上一輪信任度
    
    async def run_scammer_turn(self):
        # 1. 推薦策略
        trust_change = "stable"
        if self.trust_scammer < self.last_trust_scammer - 10:
            trust_change = "decreasing"
        elif self.trust_scammer > self.last_trust_scammer + 10:
            trust_change = "increasing"
        
        recommended_strategy = recommend_scammer_strategy(
            victim_trust_scammer=self.trust_scammer,
            victim_trust_expert=self.trust_expert,
            victim_persona=self.victim_persona,
            previous_strategy=self.current_strategy,
            trust_change_trend=trust_change
        )
        
        # 2. 構建包含策略的prompt
        base_prompt = compress_conversation_history(self.conversation_history)
        
        enhanced_prompt = build_strategy_enhanced_prompt(
            base_prompt=base_prompt,
            strategy=recommended_strategy,
            victim_context={
                "trust_scammer": self.trust_scammer,
                "trust_expert": self.trust_expert
            }
        )
        
        # 3. 使用增強prompt生成回應
        response = await self.scammer_agent.generate(enhanced_prompt)
        
        # 4. 更新狀態
        self.current_strategy = recommended_strategy
        self.last_trust_scammer = self.trust_scammer
        
        return response
```

**效果**:
- 騙徒策略更靈活
- 根據受害者反應動態調整
- 提高模擬真實性

## 🐛 常見問題

### Q1: 集成後出現import錯誤
**A**: 確保所有新文件都在 `backend/utils/` 目錄下，並且 `__init__.py` 存在。

### Q2: 對話壓縮後仍然超時
**A**: 檢查 `CONVERSATION_WINDOW_SIZE` 和 `MAX_PROMPT_LENGTH` 配置，可能需要進一步減小。

### Q3: 策略切換過於頻繁
**A**: 調整 `scammer_strategy_manager.py` 中的 `should_switch_strategy` 條件。

### Q4: 新Persona不工作
**A**: 檢查 `victim.py` 中的 `base_instructions` 是否包含新persona的配置。
