## 📈 性能測試檢查

### Prompt長度測試
創建測試文件 `test_prompt_length.py`:
```python
from utils.conversation_summarizer import ConversationSummarizer

# 模擬30輪長對話
long_conversation = [
    {"speaker": "騙徒", "dialogue": "你好，我係銀行職員..." * 50},
    {"speaker": "受騙者", "dialogue": "咩？我好驚啊！..." * 50},
] * 15

summarizer = ConversationSummarizer()

# 原始長度
original_length = summarizer.estimate_prompt_length(long_conversation)
print(f"原始長度: {original_length}")

# 壓縮後
compressed = summarizer.build_compressed_prompt(long_conversation)
print(f"壓縮後: {len(compressed)}")
print(f"壓縮率: {(1 - len(compressed)/original_length)*100:.1f}%")

assert len(compressed) < original_length * 0.5, "壓縮效果不足"
print("✅ 壓縮效果達標")
```

### 策略切換測試
```python
from utils.scammer_strategy_manager import StrategyManager

manager = StrategyManager()

# 測試多種場景
test_cases = [
    (85, 30, "elderly", "stable"),  # 高信任，穩定
    (55, 70, "average", "decreasing"),  # 信任下降
    (40, 60, "greedy", "decreasing"),  # 貪婪型
    (25, 85, "overconfident", "decreasing"),  # 過度自信
    (10, 90, "skeptical", "decreasing"),  # 多疑型
]

for scammer_trust, expert_trust, persona, trend in test_cases:
    strategy = manager.recommend_strategy(
        victim_trust_scammer=scammer_trust,
        victim_trust_expert=expert_trust,
        victim_persona=persona,
        trust_change_trend=trend
    )
    print(f"{persona:15} (Trust: {scammer_trust:2}) -> {strategy.value}")

print("✅ 策略切換正常")
```

**預期結果**:
- [ ] 不同場景推薦不同策略
- [ ] 策略選擇合理
