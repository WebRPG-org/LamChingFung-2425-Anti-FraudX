## 📋 **詳細步驟**

### 第 1 步：安裝依賴

**方法 A：使用腳本（推薦）**
```
雙擊：安裝依賴.bat
```

**方法 B：手動安裝**
```bash
cd backend
pip install -r requirements.txt
```

如果完整安裝失敗，只需安裝核心依賴：
```bash
pip install fastapi uvicorn requests python-dotenv ollama pydantic
```

---

### 第 2 步：配置環境

在 `backend/` 文件夾創建 `.env` 文件：

```env
AGENT_MODEL=gemma3:4b
OLLAMA_BASE_URL=http://127.0.0.1:11434
```

詳細說明請查看：`ENV_配置說明.txt`

---

### 第 3 步：安裝 Ollama

1. 下載 Ollama：https://ollama.com
2. 安裝後執行：
   ```bash
   ollama pull gemma3:4b
   ```
3. 確保 Ollama 運行：
   ```bash
   ollama serve
   ```

---

### 第 4 步：啟動系統

**方法 A：快速啟動（推薦）**
```
雙擊：快速啟動.bat
```

**方法 B：Python 啟動**
```bash
python start_server.py
```

**方法 C：直接啟動**
```bash
cd backend
python main.py
```

---

### 第 5 步：訪問系統

- **Web UI**：http://localhost:8000
- **API 文檔**：http://localhost:8000/docs
- **ReDoc**：http://localhost:8000/redoc

## 🎮 **使用 RPG Maker**

### 1. 複製插件

將以下文件複製到 RPG Maker 的 `js/plugins/` 文件夾：

```
rpg_maker_plugins/AI_Bridge.js
rpg_maker_plugins/AntiFraudGame.js
```

### 2. 啟用插件

1. 打開 RPG Maker MZ
2. 工具 → 插件管理器
3. 添加並啟用兩個插件

### 3. 使用插件

在事件中使用插件命令。

詳細教程請查看：`RPG_MAKER_整合指南.md`

## 🧪 **測試系統**

### 測試 Web UI

1. 訪問 http://localhost:8000
2. 選擇參數
3. 點擊「開始對話」
4. 觀察實時更新

### 測試 API

```bash
# 測試聊天 API
curl -X POST http://localhost:8000/chat \
  -H "Content-Type: application/json" \
  -d "{\"role\":\"測試\",\"message\":\"你好\",\"history\":[]}"

# 測試遊戲 API
curl -X POST http://localhost:8000/api/game/start \
  -H "Content-Type: application/json" \
  -d "{\"persona_type\":\"A\"}"
```

```

◆ 等待：30幀
◆ 條件分支：變量[2] ≠ ""
  └ 連接成功
```

## ⚠️ **常見問題**

### Q: ModuleNotFoundError?

**A:** 執行 `安裝依賴.bat`

### Q: Ollama 連接失敗?

**A:** 
1. 確保 Ollama 運行：`ollama serve`
2. 測試連接：`curl http://localhost:11434/api/tags`

### Q: 端口 8000 被占用?

**A:** 
1. 找到占用進程：`netstat -ano | findstr :8000`
2. 結束進程或修改端口

### Q: RPG Maker 無法連接?

**A:** 
1. 確保後台運行
2. 檢查插件 URL：`http://localhost:8000/chat`
3. 查看 F8 控制台錯誤
