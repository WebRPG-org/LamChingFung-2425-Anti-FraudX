## ❌ 常見問題排查

### 問題 1: ContextManager 測試失敗

**錯誤**: `ModuleNotFoundError: No module named 'utils'`

**解決**:
```bash
cd backend
python -m utils.context_manager
```

---

### 問題 2: 信任度仍然變化太快

**檢查**:
1. 確認 `backend/agents/victim.py` 已更新
2. 重啟服務器

**驗證修改**:
```bash
grep "之前+10，太快" backend\agents\victim.py
```

應該看到:
```
  * 對方展示「官方身份」：+5（之前+10，太快）
  * 對方提供「著數」「福利」：+8（之前+15，太快）
```

---

### 問題 3: Recorder 評分仍然顯示 0.0

**檢查日誌**:
```
📊 [Recorder 最終評分]
```

如果看不到這段，可能是：
1. `backend/api/simulation_routes.py` 未更新
2. Recorder 返回的 JSON 格式不正確

**解決**: 檢查 RecorderAgent 的輸出是否包含 `scammer_performance` 和 `expert_performance`

---

### 問題 4: 訓練數據沒有生成

**檢查**:
```bash
cd backend\training_data\finetuning
dir
```

**如果為空**:
1. 檢查日誌中是否有 "🔄 開始生成fine-tuning訓練數據"
2. 檢查是否有錯誤信息

**常見原因**:
- Recorder 分析格式錯誤
- 對話太短（< 2 輪）
